# 📊 Machine Learning Codes

A collection of Machine Learning algorithms implemented in **Python (Jupyter Notebook - .ipynb)**.  
This repository covers fundamental to advanced ML concepts with practical examples.

---

## 🚀 Topics Covered

- Data Preprocessing & Visualization  
- Linear Regression  
- Logistic Regression  
- Naive Bayes  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  
- Decision Tree  
- Artificial Neural Networks (ANN)  
- Convolutional Neural Networks (CNN)  
- Generative Adversarial Networks (GANs)  
- Ensemble Learning  

---

## 🛠️ Tech Stack

- Python  
- NumPy, Pandas  
- Matplotlib, Seaborn  
- Scikit-learn  
- TensorFlow / Keras  
- Jupyter Notebook  

---

## ▶️ Getting Started

```bash
git clone https://github.com/IrfanTech-X/Machine-Learning-Codes.git
cd Machine-Learning-Codes
pip install -r requirements.txt
jupyter notebook
```
